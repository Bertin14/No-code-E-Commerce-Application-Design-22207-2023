## ⚡ Challenges Faced

> *Real challenges encountered while building Rwanda Crafts Store.*

1. **🎨 Choosing the Right Design Theme** — Finding a template that matched the cultural and traditional identity of Rwanda Crafts Store was not easy. Many templates felt too modern or generic and required heavy customization to reflect the authentic Rwandan aesthetic.
2. **🖼️ Product Image Consistency** — Sourcing and resizing high-quality images for all 5 products so they appeared uniform across the product page was time-consuming and required careful formatting within the platform.
3. **🛒 Setting Up Cart Functionality** — Configuring the add-to-cart feature and making sure it worked correctly across all product listings required carefully following the platform's e-commerce settings and tutorials.
4. **📝 Writing the README in Markdown** — Learning and applying GitHub Markdown syntax for the first time — including tables, checkboxes, image embedding, and code blocks — was a new and challenging skill to master.

---

## 💡 Lessons Learned

> *Key takeaways from building Rwanda Crafts Store.*

1. **🚀 No-Code platforms are powerful** — Tools like Wix, WordPress, and Shopify allow anyone to build a professional e-commerce store without writing a single line of code, making web development accessible to all.
2. **🎯 UI/UX matters deeply** — A clean, well-organised layout builds trust with customers. Simple navigation and clear product descriptions significantly improve the shopping experience.
3. **📄 Documentation is as important as the product** — Writing a structured README in Markdown taught me how developers and designers professionally communicate and present their work to the world.
4. **🌍 GitHub is a portfolio tool** — Hosting this project on GitHub means it lives beyond this course — it is now part of my professional portfolio that future employers can see.
